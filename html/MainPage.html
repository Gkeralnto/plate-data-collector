Λίγα λόγια για το project μου.

Το project αφορά την χρήση νευρωνικού δικτύου για την αναγνώριση πινακίδων κυκλοφορίας. Παρόλο που υπάρχουν διάφορες εφαρμογές τέτοιων προγραμμάτων το συγκεκριμένο project προορίζεται για εκπαιδευτικούς σκοπούς. Ψάχνοντας κανείς πληροφορίες για τα νευρωνικά δίκτυα θα ανακαλύψει ότι η λειτουργία τους βασίζεται σε μεγάλο όγκο δεδομένων. Τα δεδομένα αυτά χρησιμοποιούνται για την "εκπαίδευση" του νευρωνικού δικτύου. Στο διαδίκτυο βρίσκονται χιλιάδες δεδομένα, αλλά τα απαιτούμενα δεδομένα για το project είναι δυσεύρετα. Δημιουργήθηκε έτσι η ανάγκη να αναπτύξω αυτήν την ιστοσελίδα όπου μπορείτε να με βοηθήσετε να συλλέξω τα δεδομένα που χρειάζομαι. Ανεβάζοντας φωτογραφίες από ελληνικές πινακίδες με βοηθάτε να δημιουργήσω μια μεγάλη βάση δεδομένων που εξυπηρετεί τον σκοπό μου. Μέσω της σελίδας αυτής μπορεί να γίνει και ταξινόμηση των δεδομένων που έχουν συλλεχθεί. Μετά την ολοκλήρωση του, το project θα είναι διαθέσιμο στο github μέσω αυτού του συνδέσμου.

Οι φωτογραφίες που μεταφορτώνονται υπόκεινται αμέσως σε επεξεργασία: Οι χαρακτήρες απομονώνονται, αποκόπτωνται και προστίθενται στην βάση δεδομένων ως ξεχωριστές φωτογραφίες. Η ανάκτηση των χαρακτήρων από τη βάση δεδομένων και η εμφάνισή τους γίνεται με τυχαίο τρόπο. Σε κανένα σημείο της διαδικασίας (πλην της μεταφόρτωσης) δεν μπορεί ο χρήστης να ανακατασκευάσει με ακρίβεια τις αρχικές εικόνες και να ανακτήσει τους αριθμούς κυκλοφορίας Ι.Χ. Δίνεται μεγάλη βάση στη διαφύλαξη των προσωπικών δεδομένων και την προστασία της ιδιωτικότητας των ιδιοκτητών.

Η ιστοσελίδα δημιουργήθηκε και είναι υπό τη διαχείριση του Γκεράλντο Μπάλλα (LinkedIn), που διατηρεί όλα τα δικαιώματά της.

Για τη συλλογή των δεδομένων έχει παρθεί πρώτα η συγκατάθεση των ιδιοκτητών. Τα δεδομένα χρησιμοποιούνται καθαρά για ερευνητικούς και εκπαιδευτικούς σκοπούς.

Οι πινακίδες κυκλοφορίας που συλλέγονται επεξεργάζονται με τυχαίο τρόπο ώστε να προστατευθεί η ιδιωτικότητα των ιδιοκτητών.
